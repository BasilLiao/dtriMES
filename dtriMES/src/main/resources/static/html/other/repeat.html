重工工單使用說明
(登記)方法: 使用沿用[原先舊]產品SN時，將資料重導登記至[新工單]上。
Step1. 輸入 [目前新]重工單
Step2. 輸入 [原先舊]產品SN
Step3. 按下 Check 進行資格檢查與鎖定
Step4. 系統帶出目前進度：登記數/總數(台) ?/?
Step5. 點擊 產品SN重工 [單台] 執行轉移
Step6. 數據成功更新。若想變更目標重工單，請按下 Unlock。
(歸還)方法: 當不小心配錯工單時，可將[目前新]產品還原至[舊工單]上。
Step1. 輸入 [目前新]產品SN 或 [目前新]製令單
Step2. 按下 產品SN歸還 或 製令SN歸還 即可完成防錯回復。
